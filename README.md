# PRODIGY_ML_05

## Task 05 - Food Recognition and Calorie Estimation

### Objective

The objective of this project is to develop a deep learning model capable of recognizing food items from images and estimating their calorie content. This helps users track dietary intake and make informed food choices.

### Dataset

Food-101 Dataset

### Technologies Used

* Python
* TensorFlow
* Keras
* NumPy
* Matplotlib
* EfficientNetB0
* Transfer Learning

### Deep Learning Model

* EfficientNetB0 (Pretrained on ImageNet)

### Project Workflow

1. Dataset Loading
2. Image Preprocessing
3. Data Visualization
4. Transfer Learning using EfficientNetB0
5. Model Training
6. Food Classification
7. Confidence Score Calculation
8. Calorie Estimation

### Features

* Recognizes food items from images
* Supports 101 food categories
* Displays prediction confidence
* Estimates calorie content
* Accepts custom uploaded images

### Results

* Total Food Categories: 101
* Model Architecture: EfficientNetB0
* Validation Accuracy: ~69%
* Successfully classified food images and estimated calorie values

### Sample Prediction

Food Item: Hamburger

Confidence: 99.16%

Estimated Calories: 295 kcal

### Learning Outcomes

* Transfer Learning
* Image Classification
* Deep Learning using TensorFlow
* Food Recognition Systems
* Calorie Estimation
* Computer Vision Applications

### Conclusion

This project demonstrates how deep learning and transfer learning techniques can be used to identify food items and estimate calorie content from images. Such systems can assist users in maintaining healthy dietary habits and tracking nutrition intake.
